  • Undergraduate Research Program
    Theme Existence and Construction of t-All-or-Nothing Transforms Over Arbitraty Alphabets
    Mentor Xiande Zhang(Prof. - Math, USTC)
    Date 2018.12.31-2019.10.24
    Detail A (t,s,v)-all-or-nothing(AONT) transforms is a bijective mapping defined on s-tuples over an alphabet of size v, which satisfies the condition that the values of any t values of any s-t output co-ordinates. This is originally defined by Rivest in this paper in the setting of cumputational security. Then Stinson considered unconditionally secure all-or-nothing transforms in this paper. I also study about the randomized AONT which have been studied in the context of "exposure resilient cryptography".
    Related See final thesis here. (You can also look at summary 1 and summary 2.


  • Image Synthesis(Class project)
    Theme Image synthesis using Graph Cuts
    Mentor Juyong Zhang(Assist. Prof. - Math, USTC)
    Date 2019.5.31-2019.6.11
    Detail In this project, I implemented an algorithm for image and video texture synthesis using Graph Cuts. In this approach, patch regions from a sample image or video are transformed and copied to the output and then stitched together along optimal seams to generate a new (and typically larger) output. In contrast to other techniques, the size of the patch is not chosen a-priori, but instead a graph cut technique is used to determine the optimal patch region for any given offset between the input and output texture.
    Related See my report here. See my code here.


  • NTU:Summer research intern
    Theme Preimage attacks on KECCAK
    Mentor Jian Guo(Assist.Prof. - SPMS, NTU)
    Date 2019.7.2-2019.9.31
    Description In this project, I study the security of round-reduced versions of the Keccak hash function family and try to carry out more powerful preimage attacks.
    Phase 1 I first studied and implemented the technique named linear structure introduced in the paper'Linear Structures: Applications to Cryptanalysis of Round-Reduced Keccak', which allows linearization of the underlying permutation of Keccak for up to 3 rounds with large number of variable spaces.
    Phase 2 I then studied the allocating approach in another paper'Preimage Attacks on Round-reduced Keccak -224/256 via an Allocating Approach'
    Phase 3 I am now trying to apply MILP to find the best way of designing preimage attacks……TO BE CONTINUED
